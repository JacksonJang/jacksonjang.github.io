DISQUS 가입하기

  1. DISQUS 사이트에 접속해서 Get Start를 눌러줍니다.
    사이트 주소 : https://disqus.com/

  2. DISQUS 에 가입합니다.

  3. 가입 후 로그인을 하게 되면, 아래와 같은 화면이 나옵니다.
    여기서 저는 github.io 에 넣을 예정이라, I Want to install Disqus on my site를 눌러주었습니다.

  4. Website Name 입력, Category 선택을 해주고 Create Site를 누릅니다.

  5. 우리는 무료를 사용할 것이기 때문에, 밑으로 스크롤해서 Basic 으로 Subscribe Now 를 클릭합니다.

  6. 저 같은 경우에는 Jekyll 테마를 사용했기 때문에, Jekyll 을 클릭 했습니다.

  7. 지킬 테마에 맞춰서 해당 부분을 수정하고, Configure 를 눌러줍시다.

  8. Website Name, Website URL, Category가 올바르게 되어 있는지 확인하고, Next를 눌러줍니다.
